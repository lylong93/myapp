# myapp

一个的基于flutter的app
（自己学习和玩耍的例子）
## 简单功能
* 首页有网络请求，后台接口使用easy-mock 模拟生成,
上拉加载，下拉刷新 (没有本地缓存以及上拉动画)
* 第二页简单canvas动画配合折叠组件
